Validating Web Forms

First name must contain only letters and be at least 3 characters long

Last name must contain only letters and be at least 3 characters long

Email must be a valid address, e.g. example@example.com

Password must be alphanumeric (@, ., _ and - are also allowed) and be between 6 - 20 characters long

Phone must be a valid phone number (should not contain +)

Bio must be only lowercase letters, underscores, hyphens and be 8 - 50 characters long
